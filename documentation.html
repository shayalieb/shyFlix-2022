Shaya's Movie database and API

Shaya Lieberman created this application which enables you to store and manage all the things that you love about movies. You can creat your database and filter it by mame, genre, actors, year, and much more. This allows you to keep track of all your favorites like what you already watched, what do you wanna watch, and much more!

Input fields to include in the development process

  • First name
  • Last name
  • Date of birth
  • Email address

Categories to include for filtering

  • Movie name
  • Year released
  • Category
  • Actors
  • Check-box if watched
  • To watch
  • Check-box if favorite movie
